# 前端暗黑星空主题重构计划

本计划将把博客前端彻底重构为沉浸式“暗黑星空”主题，移除亮色模式，并添加动态萤火虫特效。

## 1. 样式与主题重构 (`style.css`)
*   **强制暗黑模式**:
    *   移除 `:root` 中的亮色变量。
    *   将 `html.dark` 中的深色变量移动到 `:root`，使其成为默认且唯一的样式。
    *   **色彩微调**:
        *   `--text-main`: 设置为纯白 (`#ffffff`) 或极亮灰 (`#f0f0f0`)，确保在深色背景下的高对比度。
        *   `--card-bg`: 设置为高透明度的深黑 (`rgba(20, 20, 20, 0.6)`)，让背景图和萤火虫能透出来。
        *   `--border-color`: 设置为微弱的白色半透明 (`rgba(255, 255, 255, 0.1)`)。
*   **全局背景**:
    *   移除 `body` 或 `.page-bg` 的默认背景色。
    *   准备一个全屏的容器用于放置背景图和 canvas 特效。

## 2. 动态背景组件 (`BackgroundEffect.vue`)
*   **新建组件**: `frontend/src/components/BackgroundEffect.vue`。
*   **背景图层**:
    *   使用一个绝对定位的 `div` 或 `img` 作为底层背景。
    *   默认使用一张高质量的“黑夜星空”图片 (Unsplash Source)。
    *   支持用户通过 CSS 变量或代码轻松替换图片 URL。
*   **萤火虫特效 (Canvas)**:
    *   在背景图之上、内容层之下放置一个 `<canvas>`。
    *   编写 JS 逻辑生成“光点”粒子：
        *   **属性**: 随机位置 (x, y)、随机大小 (radius)、随机透明度 (opacity)、随机速度 (vx, vy)。
        *   **动画**: 使用 `requestAnimationFrame` 让光点缓慢漂浮、呼吸闪烁 (改变透明度)。
        *   **交互**: (可选) 鼠标移动时稍微排斥或吸引光点。

## 3. 布局调整 (`UserLayout.vue`)
*   **移除切换按钮**: 删除 Header 中的“太阳/月亮”切换按钮及其相关 JS 逻辑 (`toggleTheme`, `isDark`)。
*   **引入背景**: 在 `UserLayout.vue` 最外层引入并挂载 `<BackgroundEffect />` 组件，确保它覆盖整个视口且位于最底层 (`z-index: -1`)。
*   **透明化处理**: 确保 `el-container`, `el-header`, `el-main` 等容器背景均为透明，只依靠组件自身的半透明玻璃效果。

## 4. 组件适配 (`GlassCard`, `ArticleCard` 等)
*   检查现有组件的 CSS，确保它们没有硬编码的白色背景。
*   确认 `backdrop-filter` 在新背景下的显示效果，可能需要增加模糊度或调整透明度以保证文字可读性。

## 5. 执行顺序
1.  创建 `BackgroundEffect.vue`。
2.  修改 `UserLayout.vue` (移除 toggle，引入背景)。
3.  重写 `style.css` (强制暗黑变量)。
4.  验证效果并微调。
