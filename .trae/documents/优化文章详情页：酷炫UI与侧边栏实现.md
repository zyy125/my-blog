# 美化文章阅读页面与新增侧边栏计划

我们将对文章详情页 (`ArticleDetail.vue`) 进行彻底重构，使其视觉风格与主页保持一致（酷炫、玻璃拟态、动态效果），并增加功能丰富的侧边栏。

## 1. 布局重构 (Layout)
将原本的单栏布局改为 **双栏布局** (Left 18 : Right 6)，使用 `el-row` 和 `el-col` 实现。

### 左侧：沉浸式阅读区
1.  **顶部 Hero Header**:
    *   取消原本简单的标题栏，改为一个带有玻璃拟态效果的 **Header Card**。
    *   包含文章标题、发布时间、分类、阅读量等元数据，使用图标美化。
    *   如果有封面图，尝试将封面图作为背景或以大图形式优雅展示，与 Header 融合。
2.  **内容区域 (Content)**:
    *   将 Markdown 内容包裹在 **半透明玻璃卡片** (`backdrop-filter: blur`) 中。
    *   优化排版：增加行高，优化字体大小，确保在深色星空背景下的可读性。
    *   添加 `fade-in-up` 进场动画。
3.  **评论区 (Comments)**:
    *   重设计评论列表和表单，使其更像“对话气泡”或现代社交应用的风格。
    *   保持玻璃拟态风格，与上方内容区区分开。

### 右侧：功能侧边栏
1.  **目录导航 (TOC) - 新增核心功能**:
    *   实现一个 **悬浮目录** 卡片。
    *   自动解析文章内容中的 `H1-H3` 标题，生成导航列表。
    *   点击标题平滑滚动到对应位置。
2.  **作者卡片 (Profile)**:
    *   复用主页的作者卡片，展示头像、简介。
3.  **站点资讯 (Site Info)**:
    *   复用主页的站点数据展示。

## 2. 视觉风格 (Visual Style)
*   **背景**: 保持全局星空背景可见（容器透明）。
*   **玻璃拟态**: 大量使用 `rgba(255,255,255,0.7)` (亮色) 和 `rgba(30,30,30,0.6)` (深色) 配合 `backdrop-filter: blur(12px)`。
*   **边框与光效**: 为卡片添加细微的半透明边框和 Hover 发光效果。

## 3. 技术实现步骤
1.  **提取 TOC**: 编写一个 JS 函数，从 Markdown 文本中正则提取 `#` 标题，生成 `{id, text, level}` 数组。
2.  **组件拆分**: 虽然都在一个文件，但通过 CSS 类名严格区分 `article-container` 和 `sidebar`。
3.  **响应式**: 在移动端 (`xs`) 自动隐藏侧边栏，变为单栏阅读。
